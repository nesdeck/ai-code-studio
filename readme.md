# AI Chat · Code Studio

A modern AI chat interface with a built‑in code editor, real‑time credit balance, and support for multiple AI models. Built with vanilla HTML/CSS/JS and [Supabase](https://supabase.com) for authentication and data persistence.

![Screenshot](screenshot.png) <!-- you can add a screenshot later -->

## Features

- Three‑panel resizable layout (chat, editor, sidebar)
- User authentication (email/password) with Supabase
- Real‑time credit balance (each message costs credits)
- Code editor with syntax highlighting, file upload, download, and "Send to Chat"
- Model selector with pre‑defined credit costs (including ACS models)
- Code blocks in chat with Copy, Download, and Export to Editor buttons
- Dark theme with customizable accent color
- Chat history saved per user
- Fully responsive (minimum panel sizes)
